# ADR-0015 — Teaching Intelligence Engine

## Status
Accepted

## Context
Milestone 1 created the curriculum repository and 402 Canonical Teaching Objects. Milestone 2 requires Project X to understand relationships between lessons and concepts.

## Decision
Project X introduces Canonical Concept Objects (CCOs), a Relationship Engine, a Knowledge Graph, Progression Maps and an Intelligence Package generated from the curriculum package.

## Consequences
Lessons can now be understood by concept, sequence, progression and relationship rather than only by year group, unit and lesson order. Future AI, assessment, analytics and revision modules must consume the relationship graph rather than duplicating curriculum logic.
