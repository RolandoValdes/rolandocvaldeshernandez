# Project X Genesis

Project X began as a professional portfolio and evolved into a Teaching Operating System.

The major architectural breakthroughs of Milestone 1 were:

- Platform, Curriculum Package and Knowledge Package separation.
- Unified Teaching Database.
- Project X Curriculum Definition Language (PX-CDL).
- Canonical Teaching Object model.
- Digital Lesson Twin framework.
- Development Studio as the engineering headquarters.
- Privacy by Design.

Milestone 1 marks the transition from building foundations to building teaching intelligence.
