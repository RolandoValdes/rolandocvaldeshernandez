# Project X Manifesto

Project X exists to preserve, organise and continually improve professional teaching knowledge.

The platform is built on the belief that every lesson should become better every time it is taught, that every improvement should be preserved, and that technology should amplify a teacher's expertise rather than replace it.

The Platform provides the capabilities. The Curriculum provides the structure. Professional Knowledge provides the value.

Together they form a Teaching Operating System that grows alongside the educator throughout an entire career.
