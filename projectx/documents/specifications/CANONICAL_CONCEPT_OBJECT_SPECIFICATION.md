# Canonical Concept Object Specification

A Canonical Concept Object (CCO) represents a computing concept that can appear across multiple lessons, units and year groups.

## Required Fields
- CCO ID
- Name
- Domain
- Description
- Related CTOs
- Year group coverage
- Key stage coverage
- First introduced
- Mastery candidate
- Related concepts

## Purpose
CTOs represent teaching objects. CCOs represent knowledge. The Knowledge Graph is created by connecting CTOs and CCOs.
