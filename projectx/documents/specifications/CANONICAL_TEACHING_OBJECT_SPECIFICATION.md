# Canonical Teaching Object Specification

A Canonical Teaching Object (CTO) is the permanent Project X identity for a lesson.

A CTO contains:

- identity and curriculum context;
- Digital Lesson Twin shell;
- objectives and success criteria;
- resources placeholder;
- assessment placeholder;
- reflection placeholder;
- evidence placeholder;
- AI context placeholder;
- version and history.

Every lesson in the Curriculum Package must have exactly one CTO.
