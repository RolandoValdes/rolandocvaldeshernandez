# Curriculum Package Specification

A Curriculum Package is a versioned academic-year curriculum source loaded by the Project X Platform.

## Required hierarchy
Academic Year → Year Group → Scheme of Work → Unit → Lesson → Canonical Teaching Object.

## Current package
- ID: PX-CURR-2025-2026
- Academic Year: 2025–2026
- Source: PX-CDL v1.0
- Scope: Years 5–13

The Curriculum Package must not contain identifiable pupil data.
