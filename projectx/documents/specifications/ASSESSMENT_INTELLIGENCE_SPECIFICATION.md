# Assessment Intelligence Specification

Assessment Intelligence connects assessments, concepts, lessons and misconceptions.

## Core Objects
- Assessment Intelligence Package
- Assessment Record (aggregate only)
- Question Blueprint
- Misconception Signal
- Assessment Readiness Record

## Relationships
- Assessment Record -> CTO
- Assessment Record -> CCO
- Question Blueprint -> CCO
- Misconception Signal -> CCO
- Readiness Record -> CTO

## Privacy
Assessment Intelligence is explicitly aggregate/anonymised. Student identity is out of scope.
