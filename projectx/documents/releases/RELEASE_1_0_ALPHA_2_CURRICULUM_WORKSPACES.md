# Project X v4.0 — Release 1.0 Alpha.2
## Curriculum Workspaces

Release 1.0 Alpha.2 introduces the first teacher-facing workspace layer inside the Curriculum Intelligence Centre.

## Scope

- Universal workspace frame for curriculum objects
- Breadcrumb navigation
- Teaching Ribbon
- Context panel
- Unit Workspace
- Lesson Workspace / Digital Lesson Twin 2.0 shell
- Workspace tabs
- Teaching readiness indicator
- Lesson timeline
- Reflection, AI, Evidence, Assessment and History placeholders

## Design Principle

A teacher should not need to leave the Lesson Workspace to prepare, teach, assess, review or improve a lesson.

## Status

Implemented as Phase 2 of the Curriculum Intelligence Centre.
